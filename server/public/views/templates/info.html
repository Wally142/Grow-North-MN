About This Application

The Grow North Administrative Application was created by Greg Wallerus, Evan Kearney, Katie Campbell, Cam Schnackel, Mike Pettman, And Nestor Gomez Jimenez to help Grow North manage their network of volunteers and entrepenuers.

The main use of this app allows for the administrator to make and track connections between entrepenuers to help grow their business.
